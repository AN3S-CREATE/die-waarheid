# How to Use Die Waarheid

## Step 1: Install
```bash
pip install -r requirements.txt
```

## Step 2: Run
```bash
python interactive_analysis.py
```

## Step 3: Add Files

When it asks for files, enter the FULL path to each file:

**How to get file path:**
- Right-click file
- Hold Shift + right-click
- Select "Copy as path"
- Paste into program

## What Files Work
- Text files (.txt, .doc, .docx)
- Audio files (.wav, .mp3, .m4a)
- Chat exports (WhatsApp, SMS)

## What You Get
- Alerts about contradictions
- Evidence scoring (0-100)
- Timeline analysis
- Investigative checklist
